# SCD Lab 12

This repository contains my work for **SCD Lab 12**, where I practiced basic Git and GitHub operations.

## Tasks Performed

* Created a GitHub repository and cloned it to my local machine
* Added a new file, staged it, committed it, and pushed it to GitHub
* Modified the file and pushed the updated version
* Pulled updates from the remote repository after making changes on GitHub

## Purpose

The purpose of this lab is to understand how version control works using Git and how to manage projects using GitHub. It also helps in learning how to synchronize local and remote repositories.
